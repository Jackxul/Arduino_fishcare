# Arduino fishcare
A fish tank water cooling system

# System Behavior
>Temperature
+ below  7  -> danger   (Alert on) 
+ below 15  -> freezing (system stop)
+ 15 ~  18  -> exellent (system continue)
+ above 18  -> too warm (system start)
+ above 24  -> danger   (Alert On)
